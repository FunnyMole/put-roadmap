# 🚀 Путь — роадмап релизов 2026

Приложение для планирования и отслеживания релизов с задачами, статусами и тепловой картой трудозатрат.
📋 О проекте
Roadmap 2026 — это мощный инструмент для визуализации и управления этапами разработки. Позволяет создавать, редактировать и отслеживать прогресс проектов в различных режимах отображения.

✨ Ключевые возможности
📅 Режимы отображения
По месяцам — классическое представление roadmap с разбивкой по месяцам

По кварталам — агрегированное отображение по кварталам

Горизонтальный режим — детальная временная шкала с почасовой разбивкой

🔥 Тепловая карта (Treemap)
Визуализация трудозатрат в виде прямоугольников

Цветовая индикация статуса задач:

🟢 Зеленый — все задачи выполнены

🔵 Синий — задачи в работе

⚪ Серый — задачи не начаты

Размер блоков пропорционален трудозатратам

Вложенные задачи внутри этапов

📈 Диаграмма Ганта
Визуализация временных интервалов этапов

Цветная индикация статусов:

🟢 Зеленый — выполнено

🔵 Синий — в работе

⚪ Серый — не начато

🔴 Красный — просрочено

Синхронизированная вертикальная прокрутка левой и правой панелей

Горизонтальная прокрутка временной шкалы

Полоска текущей даты с круглым маркером

Масштабирование (увеличение/уменьшение)

🎯 Управление задачами
Drag-and-drop для переупорядочивания задач

Перемещение задач между этапами

Три статуса: todo, progress, done

Редактирование названий задач по клику

📊 Статистика
Общее количество этапов

Количество завершенных этапов

Этапы в работе

Всего задач и выполненных

Общий прогресс в процентах

🎨 Визуальные возможности
Цветовая индикация трудозатрат (от зеленого к красному)

Подсветка строк при наведении

Адаптивный дизайн

Мотивирующие цитаты в шапке

Сохранение в PNG

Сохранение в PDF (A4, альбомная ориентация)

Экспорт текущей активной вкладки
